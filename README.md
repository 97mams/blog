# 📘 Blog for Learn Adonis

A simple and educational blog project built with **AdonisJS**, designed
to help beginners learn the framework through practical implementation.\
This project demonstrates how to build a real-world blog with routing,
controllers, authentication, and database management.

## 🚀 Features

- ✨ User authentication (login & register)\
- 📝 Create, edit, delete blog posts\
- 🗂️ Categories and tags (optional)\
- 👤 User dashboard\
- 🔐 Protected routes\
- 📦 AdonisJS best practices\
- 🌱 Database migrations & seeders

## 📦 Tech Stack

- **AdonisJS** (Node.js MVC framework)\
- **TypeScript**\
- **Lucid ORM**\
- **MySQL**

## 📁 Project Structure

    /app
      /Controllers
      /Models
      /Validators
    /database
      /migrations
    /resources
      /views
    /start

## 🛠️ Installation

```bash
git clone https://github.com/your-username/blog-for-learn-adonis.git
```

```bash
cd blog-for-learn-adonis
```

```bash
npm install
```

```bash
cp .env.example .env
```

```bash
node ace migration:run
```

```bash
node ace serve --watch
```

App runs at http://localhost:3333
